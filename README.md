## shortURL

### 短网址生成系统
- 前后端分离 SpringBoot2.x+Vue2.x
- 前端代码在rain目录下
- 更改相关配置文件即可一键启动
- MySQL + Redis + ElementUI + Echarts

### 为什么要写
- 前前后后差不多弄了一天，结合各种框架、工具还是很简单的
- 课程的一个小作业，在细节上完善了很多
- 原理就是通过301或302重定向达到短链跳长链的效果
